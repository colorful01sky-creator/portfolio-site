# ポートフォリオサイト (Ryo's Portfolio)
自身のスキルと制作実績を紹介するためのポートフォリオサイトです。

## 🚀 公開URL
https://colorful01sky-creator.github.io/portfolio-site/

## 🛠 使用技術
- HTML5 / CSS3 (レスポンシブ対応)
- JavaScript (スクロールアニメーション、ハンバーガーメニュー)
- Google Forms (お問い合わせ機能)

## ✨ こだわったポイント
- **クライマーのアニメーション**: 趣味であるクライミングをモチーフに、スクロールに合わせてアイコンが移動するギミックをJavaScriptで実装しました。
- **レスポンシブデザイン**: PC、タブレット、スマホのそれぞれで最適なレイアウトになるよう、メディアクエリを使用して細かく調整しています。
- **保守性の高いコード**: CSSを共通パーツと各セクションで整理し、後からの修正が容易になるよう意識してコーディングしました。

## 📂 フォルダ構成
- `index.html`: メインページ
- `banner.html`: バナー作品の詳細紹介ページ
- `css/`: スタイルシートを格納
- `image/`: サイト内で使用している画像一式
